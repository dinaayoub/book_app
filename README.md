### Database Architecture
Database name: book_app
Database schema:
    table: books
    columns:
        id SERIAL PRIMARY KEY,
        author VARCHAR(255),
        title VARCHAR(255),
        isbn  VARCHAR(255),
        image_url  VARCHAR(255),
        description TEXT
